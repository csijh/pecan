Pecan

Pecan supports the development of recursive descent parsers. A grammar is developed incrementally, with testing via symbolic execution. Once the grammar has been developed, a parser can then be generated automatically from the grammar, in any programming language, as a bytecode interpreter. Alternatively, the grammar can be used to help write the parser by hand or using a combinator library.
