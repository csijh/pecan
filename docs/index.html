Pecan

Pecan supports the development of recursive descent parsers. A grammar is developed incrementally, with testing via symbolic execution. The grammar can then be compiled into a parser in almost any programming language, or it can be translated into a bytecode and interpreted. Alternatively, the grammar can simply be used to help write the parser by hand or using a combinator library.
