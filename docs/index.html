Pecan

Pecan supports the development of recursive descent parsers. A grammar can be developed incrementally, with testing via symbolic execution. The grammar can then be compiled into a parser in almost any programming language. The resulting recursive functions can be adapted or optimised by hand. Alternatively, for more dynamic applications, a grammar can be translated into a bytecode and interpreted.
